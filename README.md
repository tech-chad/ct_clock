### ct_clock

curses terminal digital clock

#### Install
``` pip install ct-clock```

#### Commands
- ```q``` quit
- ```s``` show seconds
- ```m``` 24 Hour Clock
- ```c``` Mode: normal, cycle whole
- ```b``` Blink colon
- ```e``` Show Date
- ```n``` Toggle colon
- ```d``` Reset settings to default
- ```123``` Cycle mode timing (1 every sec, 2 every min, 3 every hour)
- ```rtyuiop``` Color select
